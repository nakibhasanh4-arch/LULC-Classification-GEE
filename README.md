# LULC Classification using Google Earth Engine
This repository contains Google Earth Engine (GEE) scripts for Land Use Land Cover (LULC) classification using ESA WorldCover data.

## Project Overview
This project uses Google Earth Engine and remote sensing datasets to generate Land Use Land Cover (LULC) maps and support geospatial analysis.

## Classes

| Code | Class |
|------|--------|
| 0 | NoData |
| 2 | Water |
| 3 | Soil / Barren Land |
| 4 | Vegetation |
| 5 | Agriculture |
| 6 | Urban / Built-up |

## Platform
- Google Earth Engine (JavaScript API)
- ESA WorldCover Dataset
